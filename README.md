# ETCH A SKETCH WEB PROJECT
### An interactive website that responds to user mouse movements by changing the individual color and darkness of each square.

## The skills I have demonstrated throughout completing this:
1. How to use event listeners to listen to specific inputs such as changing grid sizes.
2. How to edit css styles using javascript like changing the color of each squares.
3. How to use template literals or `` when embedding specific values to styles.
4. How to use datasets into assigning specific values to an html tag then getting editing that value back.
5. How to check user input so that they give the correct value.
